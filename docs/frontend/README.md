## Quick Start

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

Open https://tradepulse.drivenw.com

## Key Technologies

- **SvelteKit 5** - Framework with TypeScript
- **Skeleton UI** - Component library
- **Tailwind CSS** - Styling
- **Vite** - Build tool
