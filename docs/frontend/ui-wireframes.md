# TradePulse UI Wireframes

Modern, clean interface design for trading analytics and journaling.

## Design System

### Colors
- **Primary**: Purple gradient (#667eea â†’ #764ba2)
- **Success/Profit**: Green (#10b981, #22c55e)
- **Error/Loss**: Red (#ef4444, #dc2626)
- **Neutral**: Gray scale (#6b7280, #9ca3af, #f3f4f6)
- **Background**: White / Dark (#ffffff / #1f2937)

### Typography
- **Headers**: Bold, large (text-3xl, text-2xl, text-xl)
- **Body**: Regular (text-base, text-sm)
- **Metrics**: Bold, prominent for numbers

### Spacing
- Cards: p-6 (24px padding)
- Gaps: gap-4 to gap-6 (16-24px)
- Sections: mb-6 to mb-8 (24-32px)

## Interaction Patterns

### Hover States
- **Cards**: Subtle shadow increase, slight scale (1.02)
- **Buttons**: Darken background by 10%
- **Table Rows**: Light gray background (#f9fafb)
- **Links**: Underline, color shift

### Click/Active States
- **Buttons**: Scale down (0.98), darker background
- **Checkboxes**: Checkmark with animation
- **Switches**: Slide animation (200ms ease)

### Transitions
- **Page Changes**: Fade in (300ms)
- **Modals**: Slide up from bottom (250ms)
- **Dropdowns**: Fade + slide (200ms)
- **Charts**: Animate on load (500ms)

### Micro-interactions
- **Success**: Green checkmark bounce
- **Error**: Red X shake
- **Loading**: Spinner or skeleton
- **Delete**: Fade out and collapse

---

## Accessibility

- **Keyboard Navigation**: Full tab support
- **Focus Indicators**: Visible outline on all interactive elements
- **ARIA Labels**: All icons and buttons
- **Color Contrast**: WCAG AA minimum (4.5:1 for text)
- **Screen Reader**: Descriptive labels for all content
- **Skip Links**: "Skip to main content"
